# Warrior Blood Donation Campaign

Landing page for the "Warrior Véradás" campaign.

## Features
- **Hero Section**: Responsive design with video/image background.
- **Impact Meter**: Animated counters for blood units and lives saved.
- **4 Pillars**: Informational cards.
- **Testimonials**: Carousel/Grid of donor feedback.
- **Registration Form**: Integrated with SalesAutopilot API.

## Setup
1. Clone the repository.
2. Open `index.html` in a browser (use a local server to avoid CORS issues).

## API Integration
The form communicates with SalesAutopilot.
Endpoint configuration can be found in `js/main.js`.
